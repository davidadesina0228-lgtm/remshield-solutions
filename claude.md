# RemShield — Project Rules

## Stack
- Next.js 14, App Router, TypeScript, Tailwind CSS
- Framer Motion for animations
- next-mdx-remote for blog/CMS (future)
- Space Grotesk font via next/font/google

pls make sure you are using the appropriate skill for every task

## Brand Colors (in tailwind.config.ts)
- navy: #0D1B3E (background)
- blue: #00AAFF (primary accent)
- teal: #00D4C8 (secondary accent)
- surface: #112255 (cards/surfaces)
- silver: #C8D6E5 (text/borders)

## Rules
- Never use light backgrounds
- Glassmorphism cards always
- All animations via Framer Motion whileInView
- Mobile-first, fully responsive
- "use client" only where necessary
- Build one component at a time
